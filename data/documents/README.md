# Your documents (offline)

Drop PDFs and `.txt` / `.md` files here. The agent can read them with `read_document` and answer from your own content.

- **PDF** – handbooks, reports, manuals (text is extracted locally; no internet)
- **.txt / .md** – notes, handbooks, any text you want the agent to use

You can add more files anytime; use **list_documents** in the agent to see what’s available.
